DESPRE WEBSITE

Acesta este un site web pentru AnaPower, o companie specializată în furnizarea 
de substanțe anabolice pentru cercetare științifică. Site-ul oferă informații 
despre produse, prețuri și modalități de contact pentru cercetători autorizați 
și instituții academice.

IMPORTANT: Toate produsele sunt destinate EXCLUSIV cercetării științifice și 
nu pentru consum uman.

IMPLEMENTĂRI JAVASCRIPT

1. GENERARE DINAMICĂ HTML (Cerința 8):
   
   A) Carduri Categorii (Home.html)
      • Funcția: generateCategoryCards()
      • Generează 4 carduri pentru categoriile de produse
      • Folosește document.createElement și innerHTML
      • Afișare dinamică cu icoane și descrieri

   B) Tabel Prețuri (Prices.html)  CERINȚA TABEL
      • Funcția: generatePricingTable()
      • Creează dinamic un tabel complet cu 15 produse
      • Include: nume produs, unitate, puritate, preț
      • Structură: <table>, <thead>, <tbody>, <tr>, <td>
      • Date stocate în array pricingData[]

   C) Pachete Cercetare (Prices.html)
      • Funcția: generateResearchPackages()
      • Generează 3 pachete cu prețuri și economii
      • Layout cu carduri și liste de beneficii

   D) Calculator Comenzi (Home.html)
      • Funcția: generateOrderCalculator()
      • Creează interfață calculator cu select și input
      • Include buton și zonă rezultate

   E) Formular Contact (Contact.html)
      • Funcția: generateContactForm()
      • Generează formular complet cu validare
      • Include 6 câmpuri și buton submit


2. FUNCȚII DE CALCUL (Cerința 9):
   
   A) Funcția Principală: calculateBulkPrice(quantity, unitPrice)
      • Parametri:
        - quantity: numărul de unități comandate
        - unitPrice: prețul pe unitate în EUR
      
      • Calcule:
        - Subtotal = quantity × unitPrice
        - Discount % bazat pe cantitate:
          * 5-9 unități: 5% reducere
          * 10-19 unități: 10% reducere
          * 20-49 unități: 15% reducere
          * 50-99 unități: 20% reducere
          * 100+ unități: 25% reducere
        - Discount = subtotal × (discountPercent / 100)
        - Total = subtotal - discount
      
      • Return: obiect cu { subtotal, discount, discountPercent, total }
      
      • Utilizare în 2 locații:
        1. Calculator pagina principală (Home.html)
        2. Calculator comenzi în cantitate mare (Prices.html)

   B) Event Handlers
      • setupBulkCalculator() - configurează calculatorul din Prices.html
      • Event listeners pentru butoanele "Calculează"
      • Validare input și afișare rezultate formatate

 CSS Extern (Cerința 7):
   • Tot stilul în style.css
   • Legat cu <link rel="stylesheet" href="style.css">
   • ZERO style inline (style="...")
   • ZERO embedded CSS (<style>...</style>)
   • Folosește:
     - Variabile CSS (--primary-color, etc.)
     - Box model complet
     - Grid și Flexbox pentru layout
     - Tranziții și hover effects
     - Media queries pentru responsive
